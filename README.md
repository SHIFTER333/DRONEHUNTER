# DRONEHUNTERS

Стратегія ППО на мапі України (Electron).

## Завантаження

Остання версія: [Releases](https://github.com/SHIFTER333/DRONEHUNTER/releases)

## Розробка

```bash
npm install
npm run electron
```

## Реліз

```bash
npm run dist
npm run release:upload
```

Потрібен `GH_TOKEN` з правами `repo`.
